# Blue Badge Remover v1.4.0 パッチノート

## 新機能

- 統合ダッシュボードが追加されました（非表示統計、設定、フィルター管理を一箇所で）
- 非表示にした課金バッジツイート数がリアルタイムで集計されます
- 100/500/1K/5K/10K達成時にお祝いバナーが表示されます
- トピック別フィルターパックが標準搭載されます（政治、仮想通貨/投資、攻撃的）
- 非表示ツイートに非表示理由のカテゴリが表示されます
- ツイート詳細ページでは非表示にせず赤いバナーでお知らせします
- バナーからホワイトリストに直接追加できます
- 一度展開したツイートはスクロール後も表示が維持されます
- リストページのフィルタリングが追加されました（設定で切り替え可能）

## 改善点

- ポップアップがトグル＋ミニ統計＋設定リンクに簡略化されました
- ポップアップ/ダッシュボード/オプション全体に統一デザインが適用されました
- 非表示統計がローカルタイムゾーン基準で正確に集計されます
- オプションでフィルターパックを変更すると開いているタブに即反映されます
- フィルタールールの過度なワイルドカード使用が安全に処理されます

## バグ修正

- ゴールドバッジ（企業）アカウントが誤って非表示になる問題を修正しました
- スクロール時の課金バッジツイートのちらつきを軽減しました
- フォロー中のアカウントの自己引用が非表示になる問題を修正しました
- 設定変更後に非表示ツイートが復元されない問題を修正しました
- 引用ツイートを含む詳細ページで投稿者が誤認識される問題を修正しました
- 非表示でないツイートへの不要なDOM操作を削除しました

## セキュリティ

- 外部iframeへのデータ漏洩経路をブロックしました
- 拡張機能の権限範囲を最小化しました
- HTMLインジェクションの可能性を排除しました

## インフラ

- WXTフレームワークに移行し、Chrome、Firefox、Edgeをサポートします
- 340のテストで安定性を検証しています
- GitHub Actions CI/CDで自動ビルド・デプロイします
