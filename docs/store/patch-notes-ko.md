# Blue Badge Remover v1.4.0 패치노트

## 새 기능

- 통합 대시보드가 추가되었습니다 (숨김 통계, 설정, 필터 관리를 한 곳에서)
- 숨긴 파딱 트윗 수가 실시간으로 집계됩니다
- 100/500/1K/5K/10K 숨김 달성 시 축하 배너가 표시됩니다
- 주제별 필터 팩이 기본 제공됩니다 (정치, 코인/재테크, 어그로)
- 숨겨진 트윗에 왜 숨겨졌는지 카테고리가 표시됩니다
- 트윗 상세 페이지에서는 숨기지 않고 빨간 배너로 알려줍니다
- 배너에서 바로 화이트리스트에 추가할 수 있습니다
- 한번 펼친 트윗은 스크롤 후에도 계속 보입니다
- 리스트 페이지 필터링이 추가되었습니다 (설정에서 토글)

## 개선 사항

- 팝업이 토글 + 미니 통계 + 설정 링크로 간결해졌습니다
- 팝업/대시보드/옵션 전체에 통일된 디자인 시스템이 적용되었습니다
- 숨김 통계가 로컬 타임존 기준으로 정확하게 집계됩니다
- 필터 팩을 옵션에서 활성화/비활성화하면 즉시 반영됩니다
- 필터 규칙에서 과도한 와일드카드 사용 시 안전하게 처리됩니다

## 버그 수정

- 금딱(골드뱃지) 계정이 숨겨지던 문제를 수정했습니다
- 스크롤 시 파딱 트윗이 보였다가 숨겨지는 플리커를 줄였습니다
- 팔로우 중인 파딱이 자기 트윗을 인용할 때 숨겨지던 문제를 수정했습니다
- 설정 변경 후 숨겨진 트윗이 복원되지 않던 문제를 수정했습니다
- 인용 트윗이 포함된 상세 페이지에서 작성자가 잘못 인식되던 문제를 수정했습니다
- 비파딱 트윗에 불필요한 DOM 조작이 발생하던 문제를 수정했습니다

## 보안

- 외부 iframe으로의 데이터 유출 경로를 차단했습니다
- 확장 프로그램 권한 범위를 최소화했습니다
- HTML 인젝션 가능성을 제거했습니다

- 340개 테스트로 안정성을 검증합니다
- GitHub Actions CI/CD로 자동 빌드 및 배포합니다
